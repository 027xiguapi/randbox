# miscellaneous - 杂项工具

各种实用的随机数据生成工具：

1. **[coin.mdx](/zh/docs/miscellaneous/coin)** - 抛硬币功能
2. **[dice.mdx](/zh/docs/miscellaneous/dice)** - 骰子功能（d4、d6、d8、d10、d12、d20、d30、d100）
3. **[fileWithContent.mdx](/zh/docs/miscellaneous/fileWithContent)** - 生成带有随机内容的文件
4. **[guid.mdx](/zh/docs/miscellaneous/guid)** - 生成随机 GUID（版本 4 和 5）
5. **[hash.mdx](/zh/docs/miscellaneous/hash)** - 生成随机十六进制哈希
6. **[hidden.mdx](/zh/docs/miscellaneous/hidden)** - 内部实用方法（信用卡类型、梅森旋转器、月份等）
7. **[n.mdx](/zh/docs/miscellaneous/n)** - 生成指定数量的随机项目数组
8. **[normal.mdx](/zh/docs/miscellaneous/normal)** - 正态分布随机数生成
9. **[radio.mdx](/zh/docs/miscellaneous/radio)** - 生成随机无线电呼号
10. **[rpg.mdx](/zh/docs/miscellaneous/rpg)** - RPG 骰子系统（如 3d10）
11. **[tv.mdx](/zh/docs/miscellaneous/tv)** - 生成电视台呼号（radio 的别名）
12. **[unique.mdx](/zh/docs/miscellaneous/unique)** - 生成唯一项目数组
13. **[weighted.mdx](/zh/docs/miscellaneous/weighted)** - 加权随机选择
