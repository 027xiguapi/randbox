# normal

```js
// 用法
randBox.normal()
randBox.normal({mean: 100})
randBox.normal({mean: 100, dev: 15})
```

返回一个正态分布的随机变量。

```js
randBox.normal()
=> 0.4244767651300604
```

默认情况下，这从均值为 `0` 和标准差为 `1` 开始，这是标准正态分布。

可选择指定均值和/或偏差。

```js
// 注意，由于没有指定偏差，使用默认值 `1`
randBox.normal({mean: 100})
=> 99.68352269988522

// 例如，要获得随机智商（根据定义，均值为 100，标准差为 15）
randBox.normal({mean: 100, dev: 15})
=> 85.11040121833615
```

与上述生成器结合使用，这可以是获得更真实结果的一种极其强大的方法，因为通常"纯随机"结果无法近似真实世界。
