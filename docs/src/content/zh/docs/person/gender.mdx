# gender

```js
// 用法
randBox.gender()
```

生成一个随机性别

```js
  randBox.gender();
  => 'Female'
```

可以使用可选选项参数中的 `extraGenders` 键提供额外的性别：

```js
  randBox.gender({
    extraGenders: ['Agender', 'Genderqueer', 'Trans', 'Pangender']
  });
```
