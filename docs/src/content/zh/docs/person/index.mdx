# person - 个人信息

个人信息和身份相关数据生成：

1. **[aadhar.mdx](/zh/docs/person/aadhar)** - 生成印度身份证号码（Aadhar）
2. **[age.mdx](/zh/docs/person/age)** - 生成随机年龄，支持不同类型（儿童、青少年、成人、老年人）
3. **[birthday.mdx](/zh/docs/person/birthday)** - 生成随机生日，支持多种格式和类型
4. **[cf.mdx](/zh/docs/person/cf)** - 生成意大利社会保险号码（Codice Fiscale）
5. **[cpf.mdx](/zh/docs/person/cpf)** - 生成巴西税号
6. **[first.mdx](/zh/docs/person/first)** - 生成随机名字，支持性别和国籍限制
7. **[gender.mdx](/zh/docs/person/gender)** - 生成随机性别，支持额外性别选项
8. **[last.mdx](/zh/docs/person/last)** - 生成随机姓氏，支持多种国籍
9. **[name.mdx](/zh/docs/person/name)** - 生成完整随机姓名，支持中间名、前缀、后缀等选项
10. **[prefix.mdx](/zh/docs/person/prefix)** - 生成随机姓名前缀（如 Mr.、Mrs. 等）
11. **[ssn.mdx](/zh/docs/person/ssn)** - 生成美国社会保险号码
12. **[suffix.mdx](/zh/docs/person/suffix)** - 生成随机姓名后缀（如 Jr.、Sr. 等）
13. **[zodiac.mdx](/zh/docs/person/zodiac)** - 生成随机星座
