# cpf

```js
// 用法
randBox.cpf()
```

生成一个随机巴西税号。

```js
randBox.cpf();
=> '607.116.899-62'
```

可选择省略标点符号：

```js
randBox.cpf({ formatted: false });
=> '18641718397'
```
