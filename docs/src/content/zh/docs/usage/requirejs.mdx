# requirejs

使用 <a href="http://requirejs.org">RequireJS</a> 加载 RandBox

```js
  require(['RandBox'], function(RandBox) {
    // 实例化
    var randBox = new RandBox();

    // 然后使用：
    var my_random_integer = randBox.integer();
  });
```
