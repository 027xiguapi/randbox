# cli

要在命令行中使用 RandBox，请全局安装 `randBox-cli`：

```bash
npm install -g randBox-cli
```

然后通过名称调用任何生成器，并跟上选项，如下所示：

```bash
$ randBox name --prefix true
Dr. Georgia Sanchez

$ randBox latitude --min 12.34 --max 56.78
22.01836
```

更多详情请查看 [randBox-cli README](https://github.com/randbox/randBox-cli)
