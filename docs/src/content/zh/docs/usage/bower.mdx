# bower

RandBox 可以通过 [Bower](https://bower.io) 轻松安装

```bash
bower install randBox
```

然后在你的应用程序的 HTML 中：

```html
<!-- 加载 RandBox -->
<script type="text/javascript" src="bower_components/randBox/randBox.js"></script>
<script>
  // 使用 RandBox
  alert(randBox.string());
</script>
```
