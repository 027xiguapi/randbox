# finance - 金融数据

金融相关数据生成功能：

1. **[cc.mdx](/zh/docs/finance/cc)** - 生成符合 Luhn 算法的随机信用卡号，支持指定卡类型
2. **[cc_type.mdx](/zh/docs/finance/cc_type)** - 获取信用卡类型信息，包括全名和简称
3. **[currency.mdx](/zh/docs/finance/currency)** - 生成随机货币代码或名称
4. **[currency_pair.mdx](/zh/docs/finance/currency_pair)** - 生成随机货币对
5. **[dollar.mdx](/zh/docs/finance/dollar)** - 生成随机美元金额，支持最大值设置
6. **[euro.mdx](/zh/docs/finance/euro)** - 生成随机欧元金额，支持最大值设置
7. **[exp.mdx](/zh/docs/finance/exp)** - 生成信用卡到期日期（MM/YY 格式）
8. **[exp_month.mdx](/zh/docs/finance/exp_month)** - 生成信用卡到期月份（01-12）
9. **[exp_year.mdx](/zh/docs/finance/exp_year)** - 生成信用卡到期年份，支持 2 位或 4 位格式
