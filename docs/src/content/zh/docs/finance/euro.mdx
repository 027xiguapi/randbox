# euro

```js
// 使用方法
randBox.euro()
randBox.euro({max: 250})
```

返回一个随机的欧元金额。格式取决于当前区域设置（示例使用欧洲格式显示）

```js
randBox.euro();
=> "2.560,27€"

randBox.euro();
=> "750.99€"
```

默认返回不超过 10000 的欧元金额。可以可选地指定 max 参数使其更大（或更小）。

```js
randBox.euro({max: 20});
=> "15,23€"

randBox.euro({max: 10000000})
=> "5.051.205,49€"
```
