# currency_pair

```js
// 使用方法
randBox.currency_pair()
```

生成一个货币对。便于模拟货币转换。保证返回唯一的货币对（不会返回相同的货币两次）。

```js
randBox.currency_pair();
=> [{ code: "ALL", name: "Albania Lek" }, { code: "ZWD", name: "Zimbabwe Dollar" }]
```
