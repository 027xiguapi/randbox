# exp_month

```js
// 使用方法
randBox.exp_month()
randBox.exp_month({future: true})
```

生成一个随机的信用卡到期月份。

```js
randBox.exp_month();
=> '01'
```

可以可选地指定必须是比当前月份更晚的月份。

```js
randBox.exp_month({future: true});
=> '10'
```

因此，如果在六月调用，这将返回从七月到十二月的随机月份。如果在十月调用，将返回十一月或十二月。

这是因为许多信用卡沙箱环境要求到期日期晚于当前日期，因此在生成当前年份的到期日期时，有必要生成比当前月份更晚的月份。
