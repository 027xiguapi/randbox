# shuffle

```js
// 用法
randBox.shuffle(array)
```

给定一个数组，打乱顺序并返回它。

```js
randBox.shuffle(['alpha', 'bravo', 'charlie', 'delta', 'echo']);
=> ['echo', 'delta', 'alpha', 'charlie', 'bravo']
```
