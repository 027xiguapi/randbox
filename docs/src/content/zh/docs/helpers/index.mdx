# helpers - 辅助工具

实用的辅助功能和数据操作工具：

1. **[capitalize.mdx](/zh/docs/helpers/capitalize)** - 将单词首字母大写
2. **[pick.mdx](/zh/docs/helpers/pick)** - 已弃用的选择功能，推荐使用 pickone 和 pickset
3. **[pickset.mdx](/zh/docs/helpers/pickset)** - 从数组中选择多个随机元素
4. **[pickone.mdx](/zh/docs/helpers/pickone)** - 从数组中选择一个随机元素
5. **[shuffle.mdx](/zh/docs/helpers/shuffle)** - 打乱数组顺序
6. **[pad.mdx](/zh/docs/helpers/pad)** - 数字填充功能
7. **[set.mdx](/zh/docs/helpers/set)** - 覆盖默认数据集
8. **[mixin.mdx](/zh/docs/helpers/mixin)** - 混入功能，用于扩展 RandBox
