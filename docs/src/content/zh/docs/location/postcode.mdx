# postcode

```js
// 用法
randBox.postcode()
```

生成一个随机的（英国）邮政编码。返回的邮政编码仅在邮政区域（第一个字符）和格式方面是有效的。

```js
randBox.postcode();
=> 'W6 9PF'
```
