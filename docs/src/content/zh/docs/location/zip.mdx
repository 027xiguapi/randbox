# zip

```js
// 用法
randBox.zip()
randBox.zip({plusfour: true})
```

生成一个随机的（美国）邮政编码。

```js
randBox.zip();
=> '90210'
```

可选择指定应该返回 [Zip+4][zip+4] 格式：

```js
randBox.zip({plusfour: true});
=> '01035-1838'
```

[zip+4]: http://vq.io/19rzsve
