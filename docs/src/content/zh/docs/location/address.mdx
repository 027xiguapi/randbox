# address

```js
// 用法
randBox.address()
randBox.address({short_suffix: true})
```

生成一个随机街道地址

```js
randBox.address();
=> '5447 Bazpe Lane'
```

可选择要求返回短后缀

```js
randBox.address({short_suffix: true});
=> '536 Baner Rd'
```
