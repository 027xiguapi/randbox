# country

```js
// 用法
randBox.country()
randBox.country({ full: true })
```

返回一个随机国家。

```js
randBox.country();
=> 'LT'
```

默认情况下，只返回国家的 2 字母 ISO 3166-1 代码。

可选择指定应该返回完整的国家名称。

```js
randBox.country({ full: true });
=> 'Venezuela'
```
