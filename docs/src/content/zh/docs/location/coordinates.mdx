# coordinates

```js
// 用法
randBox.coordinates()
randBox.coordinates({fixed: 2})
randBox.coordinates({format: 'dms'})
```

生成随机坐标，即纬度和经度，用逗号分隔。

```js
randBox.coordinates();
=> "-29.52974, 24.52815"
```

默认情况下，小数点后包含 5 位固定数字，可以指定其他设置。

```js
randBox.coordinates({fixed: 2});
=> "-49.16, 68.81"
```

默认情况下，坐标格式为 dd，可以指定其他格式。

```js
randBox.coordinates({format: 'ddm'});
=> "41°44.9592, 25°56.2622"
```

```js
randBox.coordinates({format: 'dms'});
=> "56°2'9.8187", 79°55'40.6812""
```
