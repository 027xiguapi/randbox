# template

返回与给定模板匹配的随机字符串。

```js
// 用法
randBox.template('{AA####}')
=> 'ZQ7803'
randBox.template('{Aa}-{##}')
=> 'Vr-78'
randBox.template('{####}:{####}:{####}')
=> '1628:5987:7803'
```

模板由任意数量的"字符替换"和"字符字面量"序列组成。"字符替换"序列以左大括号开始，包含任意数量的特殊替换字符，以右大括号结束。字符字面量可以是除大括号或反斜杠之外的任何字符。可以通过反斜杠转义在输出中包含字面量大括号或反斜杠字符。

以下替换字符可以在替换序列中使用：

   * "#"：一个随机数字
   * "a"：一个随机小写字母
   * "A"：一个随机大写字母
