# syllable

```js
// 用法
randBox.syllable()
```

返回一个半可发音的音节，2 或 3 个字母

```js
randBox.syllable();
=> 'fop'
```

音节以全小写形式返回。

音节通常在元音和辅音之间交替，用作单词的核心构建块。
