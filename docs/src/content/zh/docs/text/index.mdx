# text - 文本生成

文本和语言相关内容生成：

1. **[emoji.mdx](/zh/docs/text/emoji)** - 生成随机表情符号，支持指定类别和长度
2. **[paragraph.mdx](/zh/docs/text/paragraph)** - 生成随机段落，由半可发音的无意义单词组成，支持指定句子数量和换行选项
3. **[sentence.mdx](/zh/docs/text/sentence)** - 生成随机句子，由半可发音的无意义单词组成，支持指定单词数量
4. **[syllable.mdx](/zh/docs/text/syllable)** - 生成半可发音的音节，作为单词的构建块
5. **[word.mdx](/zh/docs/text/word)** - 生成半可发音的随机单词，支持指定音节数或长度（互斥选项）
