# RandBox 文档

RandBox 是一个功能强大的 JavaScript 随机数据生成库，提供了丰富的 API 用于生成各种类型的随机数据。本文档涵盖了 RandBox 的所有功能模块和使用方法。

## usage - 使用方法

如何在不同环境中安装和使用 RandBox：

1. **[bower.mdx](/zh/docs/usage/bower)** - Bower 包管理器安装和使用方法
2. **[browser.mdx](/zh/docs/usage/browser)** - 浏览器环境中的使用方法（简单、中级、高级）
3. **[requirejs.mdx](/zh/docs/usage/requirejs)** - RequireJS 模块加载方式
4. **[node.mdx](/zh/docs/usage/node)** - Node.js 环境中的安装和使用（npm、yarn、ES6）
5. **[function.mdx](/zh/docs/usage/function)** - 自定义随机函数的使用说明
6. **[cli.mdx](/zh/docs/usage/cli)** - 命令行工具的安装和使用方法
7. **[seed.mdx](/zh/docs/usage/seed)** - 种子功能详解（数字、字符串、多参数种子）

## basics - 基础数据类型

基础数据类型生成功能：

1. **[bool.mdx](/zh/docs/basics/bool)** - 生成随机布尔值，支持指定概率
2. **[character.mdx](/zh/docs/basics/character)** - 生成随机字符，支持字母、数字、符号等选项
3. **[falsy.mdx](/zh/docs/basics/falsy)** - 生成 JavaScript 中的虚假值
4. **[floating.mdx](/zh/docs/basics/floating)** - 生成随机浮点数，支持范围和精度设置
5. **[integer.mdx](/zh/docs/basics/integer)** - 生成随机整数，支持最小值和最大值设置
6. **[letter.mdx](/zh/docs/basics/letter)** - 生成随机字母，支持大小写控制
7. **[natural.mdx](/zh/docs/basics/natural)** - 生成自然数（非负整数）
8. **[prime.mdx](/zh/docs/basics/prime)** - 生成随机质数
9. **[string.mdx](/zh/docs/basics/string)** - 生成随机字符串，支持长度、字符池、大小写等多种选项
10. **[template.mdx](/zh/docs/basics/template)** - 基于模板生成字符串，支持占位符替换

## finance - 金融数据

金融相关数据生成功能：

2. **[cc_type.mdx](/zh/docs/finance/cc_type)** - 获取信用卡类型信息，包括全名和简称
3. **[currency.mdx](/zh/docs/finance/currency)** - 生成随机货币代码或名称
4. **[currency_pair.mdx](/zh/docs/finance/currency_pair)** - 生成随机货币对
5. **[dollar.mdx](/zh/docs/finance/dollar)** - 生成随机美元金额，支持最大值设置
6. **[euro.mdx](/zh/docs/finance/euro)** - 生成随机欧元金额，支持最大值设置
7. **[exp.mdx](/zh/docs/finance/exp)** - 生成信用卡到期日期（MM/YY 格式）
8. **[exp_month.mdx](/zh/docs/finance/exp_month)** - 生成信用卡到期月份（01-12）
9. **[exp_year.mdx](/zh/docs/finance/exp_year)** - 生成信用卡到期年份，支持 2 位或 4 位格式
