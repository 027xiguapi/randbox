# music_genre

```js
// 用法
randBox.music_genre();
```

从以下列表中返回一个随机音乐类型。

* Alternative（另类）
* Blues（蓝调）
* Children's（儿童音乐）
* Classical（古典）
* Country（乡村）
* Dance（舞曲）
* Disco（迪斯科）
* Electronic（电子）
* Folk（民谣）
* Funk（放克）
* Gospel（福音）
* Hip-Hop（嘻哈）
* Indie（独立）
* Jazz（爵士）
* Metal（金属）
* Pop（流行）
* Punk（朋克）
* R&B
* Reggae（雷鬼）
* Rock（摇滚）
* Techno（电子舞曲）
* World（世界音乐）

> 这被认为是'通用'类别。

上述所有类型都可以作为参数传入，并将返回传入类型的子类型。

```js
randBox.music_genre('hip-hop'); // 'Hyphy'
randBox.music_genre('world'); // 'Bollywood'
```

默认参数是 `general`
列表中总共有超过 600 种类型
