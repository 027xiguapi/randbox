# tempo

```js
// 用法
randBox.tempo(); // 默认在 40 到 320 之间
randBox.tempo({ min: 60, max: 120 });
```

返回一个节拍。🎵

```js
randBox.tempo();
=> 58

randBox.tempo();
=> 281

randBox.tempo({ max: 80 });
=> 45

randBox.tempo({ min: 120 });
=> 172

```
