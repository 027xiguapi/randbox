# music - 音乐

音乐相关数据生成：

1. **[music_genre.mdx](/zh/docs/music/music_genre)** - 生成随机音乐类型，包含 22 个主要类型和超过 600 个子类型
2. **[note.mdx](/zh/docs/music/note)** - 生成随机音符，支持多种选项（降音符、升音符、自然音符等）
3. **[tempo.mdx](/zh/docs/music/tempo)** - 生成随机节拍（默认 40-320 BPM）
