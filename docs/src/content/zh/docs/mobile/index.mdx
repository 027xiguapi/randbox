# mobile - 移动设备

移动设备相关标识符生成：

1. **[android_id.mdx](/zh/docs/mobile/android_id)** - 生成随机 GCM 注册 ID
2. **[apple_token.mdx](/zh/docs/mobile/apple_token)** - 生成随机 Apple 推送令牌
3. **[bb_pin.mdx](/zh/docs/mobile/bb_pin)** - 生成随机 BlackBerry 设备 PIN
4. **[wp7_anid.mdx](/zh/docs/mobile/wp7_anid)** - 生成随机 Windows Phone 7 ANID
5. **[wp8_anid2.mdx](/zh/docs/mobile/wp8_anid2)** - 生成随机 Windows Phone 8 ANID2
