# email

```js
// 用法
randBox.email()
randBox.email({domain: "example.com"})
```

返回一个具有随机域名的随机电子邮件。

```js
randBox.email()
=> 'kawip@piklojzob.gov'
```

可选择指定域名，电子邮件将是随机的，但域名不是。

```js
randBox.email({domain: 'example.com'})
=> 'giigjom@example.com'
```
