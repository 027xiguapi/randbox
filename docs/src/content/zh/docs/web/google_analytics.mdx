# google_analytics

```js
// 用法
randBox.google_analytics()
```

返回一个随机的[Google Analytics](https://support.google.com/analytics/answer/1032385?hl=en)跟踪代码。形式为 `'UA-123456-01'`

```js
randBox.google_analytics()
=> 'UA-384555-17'
```
