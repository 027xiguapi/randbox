# tld

```js
// 用法
randBox.tld()
```

从2字母国家 tld 列表加上以下集合中返回一个随机的 tld（[顶级域名][tld]）：

```js
['com', 'org', 'edu', 'gov', 'uk', 'net', 'io']
```

注意，这些不以句点开头。

[tld]: https://en.wikipedia.org/wiki/Top-level_domain
