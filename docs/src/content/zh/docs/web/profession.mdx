# profession

```js
// 用法
randBox.profession()
randBox.profession({rank: true})
```

返回一个随机职业。rank 默认为 false。

```js
randBox.profession()
=> 'Software Test Engineer'
```

可选择设置 rank 为 true 来获取带有职级的职业。

```js
randBox.profession({rank: true})
=> 'Junior Supply Chain Director'
```
