# web - 网络相关

网络和互联网相关数据生成：

1. **[avatar.mdx](/zh/docs/web/avatar)** - 生成 Gravatar 头像 URL
2. **[color.mdx](/zh/docs/web/color)** - 生成颜色，支持 6 种颜色格式（hex、rgb、rgba 等）
3. **[company.mdx](/zh/docs/web/company)** - 生成随机公司名称
4. **[domain.mdx](/zh/docs/web/domain)** - 生成随机域名
5. **[email.mdx](/zh/docs/web/email)** - 生成随机电子邮件地址
6. **[fbid.mdx](/zh/docs/web/fbid)** - 生成 Facebook ID
7. **[google_analytics.mdx](/zh/docs/web/google_analytics)** - 生成 Google Analytics 跟踪代码
8. **[hashtag.mdx](/zh/docs/web/hashtag)** - 生成随机标签
9. **[ip.mdx](/zh/docs/web/ip)** - 生成随机 IP 地址
10. **[ipv6.mdx](/zh/docs/web/ipv6)** - 生成随机 IPv6 地址
11. **[klout.mdx](/zh/docs/web/klout)** - 生成 Klout 分数
12. **[mac.mdx](/zh/docs/web/mac)** - 生成 MAC 地址
13. **[profession.mdx](/zh/docs/web/profession)** - 生成随机职业，支持职级选项
14. **[tld.mdx](/zh/docs/web/tld)** - 生成顶级域名
15. **[twitter.mdx](/zh/docs/web/twitter)** - 生成 Twitter 用户名
16. **[url.mdx](/zh/docs/web/url)** - 生成 URL，支持多种自定义选项
