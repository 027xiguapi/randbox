# weekday

```js
// 用法
randBox.weekday()
randBox.weekday({weekday_only: true})
```

返回一个星期几

```js
randBox.weekday();
=> 'Tuesday'
```

默认情况下，weekday_only 为 false。如果设置为 true，则永远不会返回星期六或星期日。
