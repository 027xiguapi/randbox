# timezone

```js
// 用法
randBox.timezone()
```

返回一个随机时区

```js
randBox.timezone();
=> {
"name": "India Standard Time",
"abbr": "IST",
"offset": 5.5,
"isdst": false,
"text": "(UTC+05:30) Chennai, Kolkata, Mumbai, New Delhi",
"utc": ["Asia/Calcutta"]
}
```
