# time - 时间日期

时间和日期相关数据生成：

1. **[ampm.mdx](/zh/docs/time/ampm)** - 生成上午或下午标识符
2. **[date.mdx](/zh/docs/time/date)** - 生成随机日期，支持多种格式和指定组件
3. **[hammertime.mdx](/zh/docs/time/hammertime)** - 生成带毫秒的 Unix 时间戳
4. **[hour.mdx](/zh/docs/time/hour)** - 生成随机小时（12 小时制或 24 小时制）
5. **[millisecond.mdx](/zh/docs/time/millisecond)** - 生成 0-999 之间的随机毫秒
6. **[minute.mdx](/zh/docs/time/minute)** - 生成 0-59 之间的随机分钟
7. **[month.mdx](/zh/docs/time/month)** - 生成随机月份，支持原始对象格式
8. **[second.mdx](/zh/docs/time/second)** - 生成 0-59 之间的随机秒
9. **[timestamp.mdx](/zh/docs/time/timestamp)** - 生成标准 Unix 时间戳
10. **[timezone.mdx](/zh/docs/time/timezone)** - 生成随机时区信息对象
11. **[weekday.mdx](/zh/docs/time/weekday)** - 生成星期几，支持工作日限制
12. **[year.mdx](/zh/docs/time/year)** - 生成随机年份，支持范围限制
