# hammertime

```js
// 用法
randBox.hammertime()
```

<p class="pullquote" data-pullquote='Stop. Hammertime.' markdown="1"></p>

生成一个随机 hammertime。

```js
randBox.hammertime();
=> 2273327300317
```

Hammertime 是对带有毫秒的 [Unix 时间](http://en.wikipedia.org/wiki/Unix_time) 的称呼。这与自 1970 年以来的毫秒数相同。它比普通的 Unix 时间戳具有更精细的粒度，因此经常用于实时应用程序。

根据创业公司的传说，Hammertime 是由一家创业公司创造的，其创始人与 M.C. Hammer 有过有趣的互动。"带毫秒的 Unix 时间"没有名称，在头脑风暴想法时（因为"带毫秒的 Unix 时间"是一个令人困惑的冗长表达），有人建议 Hammertime，它就流传了下来。
