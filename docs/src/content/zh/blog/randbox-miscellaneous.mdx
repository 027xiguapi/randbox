# RandBox 模块导览（5/12）：miscellaneous - 杂项工具

杂项模块集合概率策略、统计分布与标识符生成。更多函数参见 `/zh/docs/index` 与 `/zh/docs/miscellaneous/`。

**模块概览**
- 概率与权重：`coin/weighted/radio`
- 分布生成：`normal`、以及 `dice/rpg/tv` 等规则类生成。
- 标识与唯一性：`guid/hash/unique`、`fileWithContent`

**示例**
```ts
import RandBox from 'randbox'
const rb = new RandBox()
const side = rb.coin?.()
const val = rb.normal?.({ mean: 0, stdev: 1 })
const uid = rb.guid?.()
```

## Q&A 常见问题（FAQ）

- `weighted` 如何归一化权重？
  - 传入的权重会转换为概率分布，总和归一化为 1（视实现）。
- `normal` 是否使用 Box-Muller？
  - 常见实现采用 Box-Muller 或 Ziggurat；具体见模块文档说明。
- `unique` 如何保证不重复？
  - 通过内部集合或哈希检查；当空间耗尽需考虑冲突策略。
