# RandBox 模块导览（4/12）：location - 地理位置

位置模块涵盖经纬度、坐标、地址与电话等。更多函数参见 `/zh/docs/index` 与 `/zh/docs/location/`。

**模块概览**
- 坐标与经纬：`coordinates/latitude/longitude`
- 地址与区域：`address/city/state/country`
- 通讯与邮编：`phone/postal/postcode/zip`

**示例**
```ts
import RandBox from 'randbox'
const rb = new RandBox()
const gps = rb.coordinates?.({ precision: 6 })
const addr = rb.address?.()
const tel = rb.phone?.({ locale: 'us' })
```

## Q&A 常见问题（FAQ）

- 经纬度取值范围是什么？
  - `latitude` 通常在 `[-90, 90]`，`longitude` 在 `[-180, 180]`，可配合精度选项。
- `coordinates` 的精度如何设置？
  - 通过 `precision` 指定小数位数，如 6 位精度；注意四舍五入与格式化。
- 电话号码是否支持多地区？
  - 支持如 `us/uk/fr` 等地区格式；更多见 `phone` 文档。
