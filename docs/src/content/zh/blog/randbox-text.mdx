# RandBox 模块导览（9/12）：text - 文本生成

文本模块生成词、句与段落等自然语言片段。更多函数参见 `/zh/docs/index` 与 `/zh/docs/text/`。

**模块概览**
- 基本单位：`syllable/word`
- 组合结构：`sentence/paragraph`
- 表情与符号：`emoji`

**示例**
```ts
import RandBox from 'randbox'
const rb = new RandBox()
const words = rb.word?.({ length: 5 })
const para = rb.paragraph?.({ sentences: 3 })
```

## Q&A 常见问题（FAQ）

- 段落是否能控制句子数量？
  - 可通过 `sentences` 选项指定数量。
- 单词能否限定长度或字符集？
  - 通过选项控制长度与字符源（视实现）。
