# RandBox 模块导览（7/12）：music - 音乐

音乐模块可产生音符、速度与风格等。更多函数参见 `/zh/docs/index` 与 `/zh/docs/music/`。

**模块概览**
- 风格：`music_genre`
- 音符：`note`
- 速度：`tempo`

**示例**
```ts
import RandBox from 'randbox'
const rb = new RandBox()
const genre = rb.music_genre?.()
const bpm = rb.tempo?.({ min: 60, max: 180 })
```

## Q&A 常见问题（FAQ）

- `tempo` 的范围如何设定？
  - 通过选项传入最小与最大 BPM；范围应符合音乐常识。
- 风格是否可本地化？
  - 可通过数据源或 locale 扩展（视实现）。
