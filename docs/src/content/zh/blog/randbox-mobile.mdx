# RandBox 模块导览（6/12）：mobile - 移动设备

移动模块生成常见移动平台设备标识符。更多函数参见 `/zh/docs/index` 与 `/zh/docs/mobile/`。

**模块概览**
- Android：`android_id`
- iOS：`apple_token`
- Windows Phone：`wp7_anid`、`wp8_anid2`

**示例**
```ts
import RandBox from 'randbox'
const rb = new RandBox()
const aid = rb.android_id?.()
const ios = rb.apple_token?.()
```
