# RandBox 模块导览（11/12）：time - 时间日期

时间模块覆盖日期、时间、时间戳与时区。更多函数参见 `/zh/docs/index` 与 `/zh/docs/time/`。

**模块概览**
- 基本单位：`year/month/weekday/hour/minute/second/millisecond`
- 复合结构：`date/hammertime/timestamp`
- 时区与上午下午：`timezone/ampm`

**示例**
```ts
import RandBox from 'randbox'
const rb = new RandBox()
const when = rb.date?.({ year: { min: 1990, max: 2025 } })
const tz = rb.timezone?.()
```

## Q&A 常见问题（FAQ）

- `date` 是否支持范围约束？
  - 支持按年/月/日范围控制；详见文档。
- `timestamp` 与 `hammertime` 有何区别？
  - 不同时间戳表示或单位（视实现）；请参考具体条目。
