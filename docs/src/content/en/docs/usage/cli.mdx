# cli

To use RandBox from the command line, install `randBox-cli` globally with:

```bash
npm install -g randBox-cli
```

Then invoke any generator by name followed by options, like so:

```bash
$ randBox name --prefix true
Dr. Georgia Sanchez

$ randBox latitude --min 12.34 --max 56.78
22.01836
```

More details in the [randBox-cli README](https://github.com/randbox/randBox-cli)
