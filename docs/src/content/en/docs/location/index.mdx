# location - Geographical Location

Geographical location and address related data generation:

1. **[address.mdx](/en/docs/location/address)** - Generate a random street address
2. **[altitude.mdx](/en/docs/location/altitude)** - Generate a random altitude
3. **[areacode.mdx](/en/docs/location/areacode)** - Generate a random area code (USA)
4. **[city.mdx](/en/docs/location/city)** - Generate a random city name
5. **[coordinates.mdx](/en/docs/location/coordinates)** - Generate random coordinates (latitude and longitude)
6. **[country.mdx](/en/docs/location/country)** - Generate a random country code or full name
7. **[depth.mdx](/en/docs/location/depth)** - Generate a random depth (negative number)
8. **[geohash.mdx](/en/docs/location/geohash)** - Generate a random geohash
9. **[latitude.mdx](/en/docs/location/latitude)** - Generate a random latitude
10. **[locale.mdx](/en/docs/location/locale)** - Generate a random language code or regional code
11. **[longitude.mdx](/en/docs/location/longitude)** - Generate a random longitude
12. **[phone.mdx](/en/docs/location/phone)** - Generate a random phone number (supports USA, UK, France)
13. **[postal.mdx](/en/docs/location/postal)** - Generate a Canadian postal code
14. **[postcode.mdx](/en/docs/location/postcode)** - Generate a UK postcode
15. **[province.mdx](/en/docs/location/province)** - Generate a random province (supports Canada, Italy)
16. **[state.mdx](/en/docs/location/state)** - Generate a random state (supports USA, Italy)
17. **[street.mdx](/en/docs/location/street)** - Generate a random street name
18. **[zip.mdx](/en/docs/location/zip)** - Generate a US ZIP code
