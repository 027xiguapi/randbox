# geohash

```js
// usage
randBox.geohash()
randBox.geohash({ length: 5 })
```

Generate a random geohash. [More details on what a geohash is](http://en.wikipedia.org/wiki/Geohash).

```js
randBox.geohash()
=> 'h9xhn7y'
```

By default, includes 7 characters of accuracy. Can override with the `length` option.

```js
randBox.geohash({ length: 5 })
=> 'dr0kr'
```
