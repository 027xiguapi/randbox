# country

```js
// usage
randBox.country()
randBox.country({ full: true })
```

Return a random country.

```js
randBox.country();
=> 'LT'
```

By default, returns only the 2 letter ISO 3166-1 code for the country.

Optionally specify that it ought to return a full country name.

```js
randBox.country({ full: true });
=> 'Venezuela'
```
