# address

```js
// usage
randBox.address()
randBox.address({short_suffix: true})
```

Generate a random street address

```js
randBox.address();
=> '5447 Bazpe Lane'
```

Optionally mandate that it return a short suffix

```js
randBox.address({short_suffix: true});
=> '536 Baner Rd'
```
