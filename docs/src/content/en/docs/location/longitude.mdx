# longitude

```js
// usage
randBox.longitude()
randBox.longitude({fixed: 7})
randBox.longitude({format: 'dms'})
```

Generate a random longitude.

```js
randBox.longitude();
=> 149.41549
```

_range: -180 to 180_

By default includes 5 fixed digits after decimal, can specify otherwise.

```js
randBox.longitude({fixed: 7});
=> 51.4549925
```

By default includes entire range of allowed longitudes, can specify a min and/or max to bound it

```js
randBox.longitude({min: -78, max: -77});
=> -77.22644
```

By default longitude' format is dd, can specify otherwise.

```js
randBox.longitude({format: 'ddm'});
=> "41°44.9592"
```

```js
randBox.longitude({format: 'dms'});
=> "56°2’9.8187”"
```
