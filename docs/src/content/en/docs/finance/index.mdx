# finance - Financial Data

Financial related data generation functionality:

1. **[cc.mdx](/en/docs/finance/cc)** - Generate a random credit card number compliant with the Luhn algorithm, supports specifying card type
2. **[cc_type.mdx](/en/docs/finance/cc_type)** - Get credit card type information, including full name and abbreviation
3. **[currency.mdx](/en/docs/finance/currency)** - Generate a random currency code or name
4. **[currency_pair.mdx](/en/docs/finance/currency_pair)** - Generate a random currency pair
5. **[dollar.mdx](/en/docs/finance/dollar)** - Generate a random US dollar amount, supports maximum value setting
6. **[euro.mdx](/en/docs/finance/euro)** - Generate a random Euro amount, supports maximum value setting
7. **[exp.mdx](/en/docs/finance/exp)** - Generate a credit card expiration date (MM/YY format)
8. **[exp_month.mdx](/en/docs/finance/exp_month)** - Generate a credit card expiration month (01-12)
9. **[exp_year.mdx](/en/docs/finance/exp_year)** - Generate a credit card expiration year, supports 2-digit or 4-digit format
