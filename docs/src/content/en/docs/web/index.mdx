# web - Web Related

Web and internet related data generation:

1. **[avatar.mdx](/en/docs/web/avatar)** - Generate a Gravatar avatar URL
2. **[color.mdx](/en/docs/web/color)** - Generate a color, supports 6 color formats (hex, rgb, rgba, etc.)
3. **[company.mdx](/en/docs/web/company)** - Generate a random company name
4. **[domain.mdx](/en/docs/web/domain)** - Generate a random domain name
5. **[email.mdx](/en/docs/web/email)** - Generate a random email address
6. **[fbid.mdx](/en/docs/web/fbid)** - Generate a Facebook ID
7. **[google_analytics.mdx](/en/docs/web/google_analytics)** - Generate a Google Analytics tracking code
8. **[hashtag.mdx](/en/docs/web/hashtag)** - Generate a random hashtag
9. **[ip.mdx](/en/docs/web/ip)** - Generate a random IP address
10. **[ipv6.mdx](/en/docs/web/ipv6)** - Generate a random IPv6 address
11. **[klout.mdx](/en/docs/web/klout)** - Generate a Klout score
12. **[mac.mdx](/en/docs/web/mac)** - Generate a MAC address
13. **[profession.mdx](/en/docs/web/profession)** - Generate a random profession, supports rank options
14. **[tld.mdx](/en/docs/web/tld)** - Generate a Top-Level Domain
15. **[twitter.mdx](/en/docs/web/twitter)** - Generate a Twitter username
16. **[url.mdx](/en/docs/web/url)** - Generate a URL, supports multiple custom options
