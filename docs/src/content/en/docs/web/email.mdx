# email

```js
// usage
randBox.email()
randBox.email({domain: "example.com"})
```

Return a random email with a random domain.

```js
randBox.email()
=> 'kawip@piklojzob.gov'
```

Optionally specify a domain and the email will be random but the domain will not.

```js
randBox.email({domain: 'example.com'})
=> 'giigjom@example.com'
```
