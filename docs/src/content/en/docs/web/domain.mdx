# domain

```js
// usage
randBox.domain()
randBox.domain({tld: 'com'})
```

Return a random domain with a random [tld](#tld).

```js
randBox.domain()
=> 'onaro.net'
```

Optionally specify a tld and the domain will be random but the tld will not.

```js
randBox.domain({tld: 'ie'})
=> 'gotaujo.ie'
```
