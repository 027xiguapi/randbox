# profession

```js
// usage
randBox.profession()
randBox.profession({rank: true})
```

Return a random profession. Rank is false by default.

```js
randBox.profession()
=> 'Software Test Engineer'
```

Optionally set rank as true get rank with profession.

```js
randBox.profession({rank: true})
=> 'Junior Supply Chain Director'
```
