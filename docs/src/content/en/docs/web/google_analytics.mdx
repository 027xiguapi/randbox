# google_analytics

```js
// usage
randBox.google_analytics()
```

Return a random [Google Analytics](https://support.google.com/analytics/answer/1032385?hl=en) tracking code. Takes the form `'UA-123456-01'`

```js
randBox.google_analytics()
=> 'UA-384555-17'
```
