# syllable

```js
// usage
randBox.syllable()
```

Return a semi-speakable syllable, 2 or 3 letters

```js
randBox.syllable();
=> 'fop'
```

The syllable is returned in all lower case.

A syllable generally alternates between vowel and consonant and is used as the
core building block for a word.
