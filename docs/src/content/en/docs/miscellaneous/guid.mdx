# guid

```js
// usage
randBox.guid()
randBox.guid({version: 5})
```

Return a random guid, version 5 by default.

```js
randBox.guid();
=> 'f0d8368d-85e2-54fb-73c4-2d60374295e3'
randBox.guid({version: 4});
=> 'c71f58e3-34af-43c0-b405-2764d6947d21'
```
