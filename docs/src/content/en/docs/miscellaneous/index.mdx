# miscellaneous - Miscellaneous Tools

Various practical random data generation tools:

1. **[coin.mdx](/en/docs/miscellaneous/coin)** - Coin toss functionality
2. **[dice.mdx](/en/docs/miscellaneous/dice)** - Dice functionality (d4, d6, d8, d10, d12, d20, d30, d100)
3. **[fileWithContent.mdx](/en/docs/miscellaneous/fileWithContent)** - Generate a file with random content
4. **[guid.mdx](/en/docs/miscellaneous/guid)** - Generate a random GUID (versions 4 and 5)
5. **[hash.mdx](/en/docs/miscellaneous/hash)** - Generate a random hexadecimal hash
6. **[hidden.mdx](/en/docs/miscellaneous/hidden)** - Internal utility methods (credit card types, Mersenne Twister, months, etc.)
7. **[n.mdx](/en/docs/miscellaneous/n)** - Generate an array of a specified number of random items
8. **[normal.mdx](/en/docs/miscellaneous/normal)** - Normal distribution random number generation
9. **[radio.mdx](/en/docs/miscellaneous/radio)** - Generate a random radio call sign
10. **[rpg.mdx](/en/docs/miscellaneous/rpg)** - RPG dice system (e.g., 3d10)
11. **[tv.mdx](/en/docs/miscellaneous/tv)** - Generate a TV station call sign (alias for radio)
12. **[unique.mdx](/en/docs/miscellaneous/unique)** - Generate an array of unique items
13. **[weighted.mdx](/en/docs/miscellaneous/weighted)** - Weighted random selection
