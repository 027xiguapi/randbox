# weekday

```js
// usage
randBox.weekday()
randBox.weekday({weekday_only: true})
```

Return a weekday

```js
randBox.weekday();
=> 'Tuesday'
```

By default, weekday_only is false. If set to true it will never return Saturday or Sunday.
