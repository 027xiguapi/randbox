# time - Date and Time

Date and time related data generation:

1. **[ampm.mdx](/en/docs/time/ampm)** - Generate AM or PM identifier
2. **[date.mdx](/en/docs/time/date)** - Generate a random date, supports multiple formats and specified components
3. **[hammertime.mdx](/en/docs/time/hammertime)** - Generate a Unix timestamp with milliseconds
4. **[hour.mdx](/en/docs/time/hour)** - Generate a random hour (12-hour or 24-hour format)
5. **[millisecond.mdx](/en/docs/time/millisecond)** - Generate a random millisecond between 0-999
6. **[minute.mdx](/en/docs/time/minute)** - Generate a random minute between 0-59
7. **[month.mdx](/en/docs/time/month)** - Generate a random month, supports raw object format
8. **[second.mdx](/en/docs/time/second)** - Generate a random second between 0-59
9. **[timestamp.mdx](/en/docs/time/timestamp)** - Generate a standard Unix timestamp
10. **[timezone.mdx](/en/docs/time/timezone)** - Generate a random timezone information object
11. **[weekday.mdx](/en/docs/time/weekday)** - Generate a weekday, supports limiting to weekdays
12. **[year.mdx](/en/docs/time/year)** - Generate a random year, supports range limits
