# basics - Basic Data Types

Basic data type generation functionality:

1. **[bool.mdx](/en/docs/basics/bool)** - Generate a random boolean value, supports specifying probability
2. **[character.mdx](/en/docs/basics/character)** - Generate a random character, supports options like letters, numbers, symbols
3. **[falsy.mdx](/en/docs/basics/falsy)** - Generate JavaScript falsy values
4. **[floating.mdx](/en/docs/basics/floating)** - Generate a random floating-point number, supports range and precision settings
5. **[integer.mdx](/en/docs/basics/integer)** - Generate a random integer, supports minimum and maximum settings
6. **[letter.mdx](/en/docs/basics/letter)** - Generate a random letter, supports case control
7. **[natural.mdx](/en/docs/basics/natural)** - Generate a natural number (non-negative integer)
8. **[prime.mdx](/en/docs/basics/prime)** - Generate a random prime number
9. **[string.mdx](/en/docs/basics/string)** - Generate a random string, supports multiple options like length, character pool, case
10. **[template.mdx](/en/docs/basics/template)** - Generate a string based on a template, supports placeholder replacement
