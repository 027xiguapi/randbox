# prime

```js
// usage
randBox.prime()
randBox.prime({ min: 1, max: 20 })
```

Return a prime number.

_default range: 0 to 10000_

```js
  randBox.prime();
  => 929
```

Can optionally provide min and max.

```js
randBox.prime({min: 1, max: 20});
=> 13
```


These are inclusive, so they are included in the range. This means
```randBox.prime({min: 2, max: 5});``` would return either 2, 3, or 5 or:

```js
// Specific case
2 <= random number <= 5

// General case
min <= random number <= max
```
