# helpers - Helper Tools

Practical helper functions and data manipulation tools:

1. **[capitalize.mdx](/en/docs/helpers/capitalize)** - Capitalize the first letter of a word
2. **[pick.mdx](/en/docs/helpers/pick)** - Deprecated selection function, recommend using pickone and pickset
3. **[pickset.mdx](/en/docs/helpers/pickset)** - Select multiple random elements from an array
4. **[pickone.mdx](/en/docs/helpers/pickone)** - Select one random element from an array
5. **[shuffle.mdx](/en/docs/helpers/shuffle)** - Shuffle the order of an array
6. **[pad.mdx](/en/docs/helpers/pad)** - Number padding functionality
7. **[set.mdx](/en/docs/helpers/set)** - Override default data sets
8. **[mixin.mdx](/en/docs/helpers/mixin)** - Mixin functionality, used to extend RandBox
