# animal

```js
// usage
randBox.animal()
randBox.animal({type: 'zoo'})
```

Generate a random animal

```js
randBox.animal();
=> 'Cobra'
```

Default is any type of animal.

Optionally specify a specific type of animal

```js
randBox.animal({type: 'zoo'});
=> 'Lion'
```

Allowed types are: `ocean`, `desert`, `grassland`, `forest`, `farm`, `pet`, and `zoo`
