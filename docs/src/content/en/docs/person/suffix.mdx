# suffix

```js
// usage
randBox.suffix()
randBox.suffix({ full: true })
```

Generate a random name suffix

```js
randBox.suffix();
=> 'Sr.'
```

By default, returns the shorter version.

Optionally get back the full version.

```js
randBox.suffix({ full: true });
=> 'Juris Doctor'
```

*To maintain legacy support, this also responds to randBox.name_suffix().*
