# gender

```js
// usage
randBox.gender()
```

Generate a random gender

```js
  randBox.gender();
  => 'Female'
```

Extra genders can be provided using the `extraGenders` key in the optional options argument:

```js
  randBox.gender({
    extraGenders: ['Agender', 'Genderqueer', 'Trans', 'Pangender']
  });
```
