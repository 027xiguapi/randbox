# person - Personal Information

Personal information and identity-related data generation:

1. **[aadhar.mdx](/en/docs/person/aadhar)** - Generate an Indian identity number (Aadhar)
2. **[age.mdx](/en/docs/person/age)** - Generate a random age, supports different types (child, teen, adult, senior)
3. **[birthday.mdx](/en/docs/person/birthday)** - Generate a random birthday, supports multiple formats and types
4. **[cf.mdx](/en/docs/person/cf)** - Generate an Italian social security number (Codice Fiscale)
5. **[cpf.mdx](/en/docs/person/cpf)** - Generate a Brazilian tax number
6. **[first.mdx](/en/docs/person/first)** - Generate a random first name, supports gender and nationality limits
7. **[gender.mdx](/en/docs/person/gender)** - Generate a random gender, supports additional gender options
8. **[last.mdx](/en/docs/person/last)** - Generate a random last name, supports multiple nationalities
9. **[name.mdx](/en/docs/person/name)** - Generate a full random name, supports middle name, prefix, suffix options
10. **[prefix.mdx](/en/docs/person/prefix)** - Generate a random name prefix (e.g., Mr., Mrs.)
11. **[ssn.mdx](/en/docs/person/ssn)** - Generate a US Social Security Number
12. **[suffix.mdx](/en/docs/person/suffix)** - Generate a random name suffix (e.g., Jr., Sr.)
13. **[zodiac.mdx](/en/docs/person/zodiac)** - Generate a random zodiac sign
