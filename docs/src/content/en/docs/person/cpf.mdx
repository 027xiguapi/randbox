# cpf

```js
// usage
randBox.cpf()
```

Generate a random Brazilian tax id.

```js
randBox.cpf();
=> '607.116.899-62'
```

Optionally omit punctuation:

```js
randBox.cpf({ formatted: false });
=> '18641718397'
```
