# mobile - Mobile Devices

Mobile device identifier generation:

1. **[android_id.mdx](/en/docs/mobile/android_id)** - Generate a random GCM registration ID
2. **[apple_token.mdx](/en/docs/mobile/apple_token)** - Generate a random Apple Push Token
3. **[bb_pin.mdx](/en/docs/mobile/bb_pin)** - Generate a random BlackBerry device PIN
4. **[wp7_anid.mdx](/en/docs/mobile/wp7_anid)** - Generate a random Windows Phone 7 ANID
5. **[wp8_anid2.mdx](/en/docs/mobile/wp8_anid2)** - Generate a random Windows Phone 8 ANID2
