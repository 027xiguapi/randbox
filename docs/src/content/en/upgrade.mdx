# 🎲 RandBox Full Feature Overview

RandBox is a powerful JavaScript random data generation library, offering over 100 data types across 10+ major categories to meet your diverse needs in development, testing, and data simulation\!

<Callout emoji={<span className='icon-[game-icons--dice-twenty-faces-twenty]'></span>} type="info">
Powerful Features

RandBox provides a rich API and flexible configuration options, supporting comprehensive random data generation from basic data types to complex business scenarios. Whether for unit testing, data population, or rapid prototyping, RandBox offers a reliable solution.
</Callout>

### 🎯 Usage Methods Module

RandBox supports multiple environments and usage methods, making integration into various projects easy:

  - 🌐 **Browser Environment**: Supports simple, intermediate, and advanced usage to meet different complexity needs.
  - 📦 **Package Managers**: Supports installation via mainstream package managers like Bower, npm, and yarn.
  - 🔧 **Module Loading**: Compatible with various loading methods, including RequireJS and ES6 modules.
  - 🖥️ **Node.js Environment**: Perfect support for server-side JavaScript development.
  - ⚙️ **Custom Functionality**: Supports custom random functions and a CLI command-line tool.
  - 🌱 **Seed System**: Supports number, string, and multi-parameter seeds to ensure reproducible randomness.

### 🎨 Basic Data Types

Core random data generation features, covering all basic data types:

  - 🔢 **Numeric Types**: Boolean, Integer, Floating-point, Natural Number, Prime Number
  - 🔤 **Character Types**: Random Character, Letter, String, Template String
  - 📝 **Special Types**: JavaScript Falsy values, Hexadecimal, Buffer

### 💰 Financial Data Module

Professional financial data generation, compliant with real-world standards:

  - 💳 **Credit Card System**: Luhn-compliant credit card numbers, CVV, expiration dates
  - 🏦 **Currency System**: Random currency codes, currency pairs, USD/Euro amounts
  - 🌐 **International Standards**: Supports IBAN and various credit card type identification

### 🛠️ Helper Tools Module

Practical data manipulation and processing tools:

  - 🎲 **Array Operations**: Random selection, shuffling, unique value generation
  - 🔧 **String Utilities**: Capitalization, number padding, weighted selection
  - ⚡ **Performance Optimization**: Mixin functionality, dataset overriding, batch generation

### 🌍 Geographical Location Module

Global geographical data generation, supporting multiple countries and regions:

  - 🏠 **Address System**: Street address, City, Country, Coordinates
  - 📞 **Communication Info**: Phone numbers, Area codes (supports USA, UK, France)

### 🎮 Miscellaneous Tools Module

A wealth of practical utilities and gamified features:

  - 🎯 **Random Selection**: Coin toss, Dice system (d4 to d100)
  - 🆔 **Identifier Generation**: GUID, Hash value, Timestamp
  - 📊 **Statistical Features**: Normal distribution, RPG dice system, Radio call signs

### 📱 Mobile Device Module

Dedicated identifier generation for mobile platforms:

  - 🤖 **Android Platform**: GCM Registration ID, Device Identifier

  - 📱 **Other Platforms**: BlackBerry PIN, Windows Phone ANID

### 🎵 Music Data Module

Professional music-related data generation:

  - ⏱️ **Tempo System**: Random tempo (40-320 BPM range)

### 👤 Personal Information Module

Comprehensive personal identity data generation, supporting multi-national standards:

  - 🆔 **Identity Documents**: US SSN, Indian Aadhar, Italian CF, Brazilian CPF

Intelligent text content generation:

### 🐾 Entities Module

Random generation of various entities:

### ⏰ Date and Time Module

A complete time data generation system:

  - 🕐 **Time Components**: Hour, Minute, Second, Millisecond, AM/PM indicator

## Quick Start

### Install RandBox

```bash
# Using npm
npm install randbox

# Using yarn
yarn add randbox

# Using bower
bower install randbox
```

### Basic Usage Example

```javascript
import RandBox from 'randbox';

const randBox = new RandBox();

// Basic Data Types
console.log(randBox.bool());           // true/false
console.log(randBox.integer(1, 100));  // 42
console.log(randBox.string(10));       // "aBcD1234eF"

// Personal Information
console.log(randBox.name());           // "Sarah Johnson"
console.log(randBox.email());          // "john.doe@example.com"
console.log(randBox.age());            // 25

// Geographical Location
console.log(randBox.city());           // "New York"
console.log(randBox.country());        // "United States"
console.log(randBox.coordinates());    // "40.7128, -74.0060"

// Financial Data
console.log(randBox.cc());             // "4111111111111111"
console.log(randBox.currency());       // "USD"
```

### Using a Seed for Reproducibility

```javascript
import RandBox from 'randbox';

// Use the same seed to generate the same result
const randBox1 = new RandBox(12345);
const randBox2 = new RandBox(12345);

console.log(randBox1.name()); // "Alice Smith"
console.log(randBox2.name()); // "Alice Smith" (Same result)
```
