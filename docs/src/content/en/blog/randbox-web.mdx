# RandBox Module Tour (12/12): web – Network and Social

**Overview**
- Identifiers and addresses: `domain/url/email/ip/ipv6/mac/tld`
- Brand/company: `company/avatar/profession`
- Social and analytics: `twitter/hashtag/fbid/google_analytics/klout`

**Examples**
```ts
import RandBox from 'randbox'
const rb = new RandBox()
const link = rb.url?.({ protocol: 'https' })
const mail = rb.email?.({ domain: 'example.com' })
const ip4 = rb.ip?.()
```

## FAQ

- Control protocol and path in `url`?
  - Use options for protocol and path structure.
- Bind custom domain in `email`?
  - Pass `domain` in options.
