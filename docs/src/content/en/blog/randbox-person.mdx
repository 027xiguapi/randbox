# RandBox Module Tour (8/12): person – Identity and Documents

**Overview**
- Names: `first/last/name`, `prefix/suffix`
- Age and birthday: `age/birthday/zodiac`
- Documents: `ssn/cpf/cf/aadhar`

**Examples**
```ts
import RandBox from 'randbox'
const rb = new RandBox()
const full = rb.name?.({ nationality: 'en' })
const born = rb.birthday?.({ year: { min: 1980, max: 2010 } })
const ssn = rb.ssn?.()
```

## FAQ

- Are documents real?
  - No. They are synthetic for testing only.
- Different languages for names?
  - Use `nationality/locale` where available.
